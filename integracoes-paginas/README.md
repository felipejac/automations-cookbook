# Integracoes Paginas

Integration guide pages - automatically generated from scraper data.

## Note

The HTML files in this folder are generated by:
- `scripts/generate_tutorials.py` - Main HTML generator
- `scripts/fix_pages.py` - Removes vote buttons and updates status messages
- `scripts/scraper_avancado.py` - Scrapes integration data from the web

## Button Status

✓ "Ver Todos os Templates" button - WORKING (redirects to /integracoes)
✗ "Votar neste Template" button - REMOVED (deleted from all pages as requested)
✓ No "Template em Desenvolvimento" message (cleaned from all pages)

## Usage

These pages are deployed to Netlify and served from the root domain at `/integracoes-paginas/`
